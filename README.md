# inclinio-admin

Panell d'administració d'Inclinio v2 (SPA Vue 3 + Vite).

Es comunica amb `inclinio-api` via cookies Sanctum.

## Convencions
- **Codi en anglès**, **comentaris en català**.
- Estructura `src/modules/<context>/` paral·lela als mòduls del backend.

## Comandes
Sempre des de `inclinio-infra/`:
```bash
./bin/admin npm install
./bin/admin npm run dev
```
